SCM2020 V1.0

Primeiro de tudo é necessário ter usuários necessários para a administração do sistema.
Dito isso, para que o sistema seja operacional é necessário que tenha as movimentações de entrada e saída dos produtos.
Para ocorrer movimentação dos produtos, é necessário cadastrar produtos.
Para tal, será cadastrado o produto (genericamente) e se caso o produto houver alguma especificação como número de patrimônio, será adicionado individualmente.
E para adicionar o produto, é necessário primeiramente cadastrar o fornecedor em questão.
